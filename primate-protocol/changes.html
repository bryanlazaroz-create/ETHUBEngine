Release Notes

Primate Protocol: Containment Sweep

A complete refactor to a Phaser 3 game loop, a pure JavaScript engine core, and Convex persistence.

Back to game

Gameplay Engine

  • Pure JavaScript logic layer with deterministic updates.
  • Vector math helpers and capture mechanics in the engine.
  • Unit tests covering movement, capture, and win conditions.

Rendering and Scenes

  • Phaser 3 boot and game scenes with real graphics.
  • HUD updates for score, remaining targets, and time.
  • Custom sprites and tiled background assets.

Persistence

  • Convex schema for profiles and save slots.
  • Autosave hooks for periodic and event-driven saves.
  • Configure the Convex URL with VITE_CONVEX_URL in .env.

Tooling

  • Vite for development and production builds.
  • ESLint for static checks.
  • Vitest for engine unit tests.
