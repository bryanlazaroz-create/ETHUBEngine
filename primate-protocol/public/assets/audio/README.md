This folder holds Mixkit audio assets for the game.

Download the sounds from https://mixkit.co/free-sound-effects/game/ and a long
Mixkit music track, then save them with the filenames below. These filenames
are referenced in `lib/game/audio.ts`.

Background music (looped)
- `music-canopy-pursuit.mp3`: Long, continuous game music (ambient/sci-fi).

Creature sound effects
- `sfx-skitterling-chirp.wav`: Quick, skittery creature chirp.
- `sfx-skitterling-stunned.wav`: Light stun zap/squeak.
- `sfx-skitterling-captured.wav`: Capture stinger/impact.
- `sfx-bark-bouncer-hop.wav`: Heavier hop/thump with ape-like tone.
- `sfx-bark-bouncer-stunned.wav`: Deep stun effect.
- `sfx-bark-bouncer-captured.wav`: Heavy capture impact.
- `sfx-inkspawn-gurgle.wav`: Wet/inky gurgle.
- `sfx-inkspawn-stunned.wav`: Watery stun snap.
- `sfx-inkspawn-captured.wav`: Inky capture splash.
- `sfx-bolt-runner-burst.wav`: Quick electric dash.
- `sfx-bolt-runner-stunned.wav`: Crackle stun.
- `sfx-bolt-runner-captured.wav`: Power-down capture hit.

Gadget sound effects
- `sfx-capture-lasso.wav`: Short tether snap.
- `sfx-pulse-baton.wav`: Electric baton hit.
- `sfx-sonic-beacon.wav`: Wide ping or sonar tone.
- `sfx-grapple-line.wav`: Grapple launch zip.

UI sound effects
- `sfx-ui-pause-open.wav`: Soft UI open chime.
- `sfx-ui-pause-close.wav`: Soft UI close chime.
